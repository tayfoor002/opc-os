# Changelog

## 0.12.1 — Enterprise Tasks

- Added task search, status, priority, owner and activity filters.
- Added sorting by due date, priority, title and creation date.
- Added live statistics for open, active, blocked, overdue and completed tasks.
- Added a right-side task details drawer for create and edit.
- Added a safe delete confirmation dialog.
- Added inline status updates and Realtime synchronization.
- No database migration required beyond the existing `public.tasks` table.
