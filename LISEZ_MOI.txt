OPC OS — Sprint 14.2 — Activity Workspace

1. Arrête l'application avec Ctrl+C.
2. Copie le dossier src dans la racine du projet et accepte le remplacement.
3. Copie le fichier supabase/sprint_014_2_activity_workspace.sql dans ton projet.
4. Dans Supabase > SQL Editor > New query, colle le contenu du fichier SQL et clique Run.
5. Relance avec npm run dev depuis CMD.
6. Ouvre Planning et clique sur une activité.

Nouveautés :
- Activity Workspace avec onglets Synthèse, Tasks, Materials, Documents, Avancement et Reporting.
- Création rapide de tâches, matériaux, documents et mises à jour d'avancement.
- Toutes les données sont reliées par activity_id.
